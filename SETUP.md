# Playwright Automation - Separate Project Setup

This folder contains all the files needed to create the standalone Playwright automation project.

## How to Use

Run the PowerShell script to create the project:

```powershell
.\create-project.ps1
```

Or manually copy the files to `E:\Norton-Gauss\playwright-automation\`

## Files Included

| File | Description |
|------|-------------|
| `package.json` | Dependencies |
| `server.js` | Express + Playwright automation |
| `Dockerfile` | Docker config for Render |
| `render.yaml` | Render deployment config |
| `.gitignore` | Git ignore |
| `README.md` | Project documentation |
